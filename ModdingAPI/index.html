Modding Setup

To create mods for GoreBox, you need:

Important: Your main script must be named hook.lua and located in the D3Mods folder in the game directory.

File Structure

📁 GameDirectory/
├──📄 GoreBox.exe # Game executable
├── 📁 D3Mods/
│   └── 📄 hook.lua   # Main mod script

Script Reloading

To reload Lua scripts during gameplay:

  • Press F3 to Recompile
  • Press F2 to Unload DLL

Core API Functions

The API provides many functions for interacting with the game world.

AutoRegister()

Enables automatic registration of game objects for subsequent manipulation.

            

              Using AutoRegister
              
            

            -- Enable player auto-registration
AutoRegister()  -- No parameters

-- All created players will be registered with their username

Instantiate(name, path, x, y, z, rx, ry, rz, rw, group)

Creates a new game object at specified coordinates.

Parameter Type Description
name string Unique object name
path string Resource path (e.g., "physcube")
x, y, z number Position coordinates
rx, ry, rz, rw number Rotation parameters (quaternion)
group integer Object group (usually 0) 
            

              Object creation example
              
            

            -- Create a cube at coordinates (0, 100, 0)
Instantiate("MyCube", "physcube", 0, 100, 0, 0, 0, 0, 0, 0)

RegisterGameObject(name, newName)

Registers a game object under a new name for subsequent access.

            

              Object registration example
              
            

            -- Register player under a new name
if sender == "Developer" then
  RegisterGameObject("Developer-PLAYER", "OMG")
end

DestroyGameObject(name)

Destroys a game object by its name.

            

              Object destruction example
              
            

            if prop == "RPG" then
  DestroyGameObject(prop)
end

SetPosition(name, x, y, z)

Sets a new position for a game object.

GetPosition(name)

Returns the current position of a game object (x, y, z).

            

              Position manipulation example
              
            

            if prop == "cube" then
  -- Move the cube to a new position
  SetPosition("cube", 100, 0, 50)
  
  -- Get current cube position
  local x, y, z = GetPosition("cube")
end

PlayerTeleport(x, y, z)

Teleports the player to specified coordinates.

            

              Player teleport example
              
            

            -- Teleport player to point (0, 50, 0)
PlayerTeleport(0, 50, 0)

Event System

The API provides event handlers to respond to game situations.

OnChatMessage(sender, message)

Called when a new chat message is received.

            

              Chat message handling example
              
            

            function OnChatMessage(sender, message)
  if message == "cube" then
    -- Create a cube when receiving "cube" command
    Instantiate("NewCube", "physcube", 0, 100, 0, 0, 0, 0, 0, 0)
  end
end

OnInstantiate(prop, player)

Called when a new game object is created.

            

              Object creation handling example
              
            

            function OnInstantiate(prop, player)
  if prop == "cube" then
    -- Move the cube when it's created
    SetPosition(prop, 100, 0, 50)
  end
  
  if prop == "RPG" then
    -- Destroy RPG when created
    DestroyGameObject(prop)
  end
end

Practical Examples

Ready-made solutions for your mods.

Chat Command System

            

              chat_commands.lua
              
            

            function OnChatMessage(sender, message)
  if message == "!teleport" then
    -- Teleport player to origin
    PlayerTeleport(0, 50, 0)
    
  elseif message == "!cube" then
    -- Create a cube above player
    local x, y, z = GetPosition("Player")
    Instantiate("CommandCube", "physcube", x, y+100, z, 0,0,0,0,0)
    
  elseif message == "!clean" then
    -- Remove all created cubes
    for i = 1, 20 do
      DestroyGameObject("CommandCube_" .. i)
    end
  end
  
end

log(I, "Chat command system loaded!")

Advanced Object Management

            

              object_manager.lua
              
            

            -- Enable object auto-registration
AutoRegister() 
-- Create table to store players
Players = {} 

function OnInstantiate(prop, player)
  -- If a player is created, add to Players table
  if prop == player then
    table.insert(Players, player)
  end
  
  -- If grenade created by D3LV!N
  if prop:find("Grenade") and player == "D3LV!N" then
    if #Players > 0 then
      -- Select random player
      local randomIndex = math.random(1, #Players)
      local x, y, z = GetPosition(Players[randomIndex])
      -- Teleport grenade to player
      SetPosition(prop, x, y, z)
    end
  end
  
  -- If money pile created by D3LV!N
  if prop:find("BigBux") and player == "D3LV!N" then
    if #Players > 0 then
      -- Select random player
      local randomIndex = math.random(1, #Players)
      local x, y, z = GetPosition(Players[randomIndex])
      -- Teleport money to player
      SetPosition(prop, x, y, z)
    end
  end
end

log(I, "Object manager activated")

Note: This example uses several important concepts:

  • Player auto-registration with AutoRegister()
  • Collecting players in Players table
  • Filtering objects by name with prop:find()
  • Teleporting objects to random players
  • Checking object creator with player == "D3LV!N"
